# 📚 Bookends AI Book Recommendation System

An intelligent book recommendation system with FAQ chatbot, built with Streamlit and Machine Learning.

## ✨ Features

- **Smart Book Recommendations**: Based on genre, title similarity, and vibe/description
- **FAQ Chatbot**: Instant answers to common questions about store policies, delivery, etc.
- **Interactive Dashboard**: Visual insights about book collection
- **User-friendly Interface**: Clean, modern design
